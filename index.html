HTTP Request from inside WASM using Wonkey's httprequest library

Actual code:

Local request := New HttpRequest
request.ReadyStateChanged = Lambda()
  If request.ReadyState = ReadyState.Done Then
    Print "Body: " + request.ResponseText
  End
End

request.Open("GET", "https://httpbin.org/anything")
request.Send()
Demo source code, Versions used on the moment of build: Wonkey 2022.04, Emscripten: 3.1.45. Created for (wannabe-awesome) list
